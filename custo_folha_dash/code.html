Lógica do Headcount

Headcount Ativo = estoque de pessoal vivo ao fim do período analisado. Uma pessoa conta quando Admissão ≤ Fim do Período e o desligamento está vazio (ou ocorre depois do fim).

loginAdmissões e logoutDesligamentos contam os eventos cuja data cai dentro do intervalo filtrado. Sem filtros, o período é a base consolidada inteira; com filtros de competência/empresa/CC, tudo é recalculado para o recorte.
